# Turtle PID Simulator

A Python-based project that demonstrates PID (Proportional-Integral-Derivative) control using Python's turtle graphics library to achieve smooth and controlled movement of a Python turtle to a user-defined pose. This project also includes a tic tac toe game, also using Python's turtle.

## Features

- PID position control demonstration
- Interactive turtle simulations
- Tic Tac Toe game
- [Web interface](https://nosh-h.github.io/Turtle-Pose-Simulator/) for project demonstration

## Technologies Used

- Python Turtle Graphics
- HTML/CSS

## Getting Started

1. Clone this repository
2. Ensure you have Python installed on your system
3. Run `pidcontrol.py` to see the PID control demonstration

## Project Structure

- `index.html` - Web interface for project demonstration
- `style.css` - Styling for the web interface
- `pidcontrol.py` - Source code for PID control demonstration with turtle graphics
- `tictactoe.py` - Source code for tic tac toe with turtle graphics
